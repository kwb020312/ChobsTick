# 🥢ChobsTick

[체험하러 가기](https://chobs-tick.vercel.app/)
🥢사업 준비를 위한 피드백 아키텍쳐 구현
필요한 의견만을 콕콕 짚어주세요

---

### 🤔어떤 역할을 하나요?

- `의견 반영`: 프로젝트를 이용하며 불편했던 점, 개선해야할 점들을 알려주세요! 빠르게 반영해드릴게요
---

### 😊어떻게 실행하나요?

필요모듈 설치
```
npm install
```

로컬환경에 실행 (localhost:3000)
```
npm run dev
```
